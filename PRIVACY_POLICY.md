# Privacy Policy / 隐私政策

**Effective date / 生效日期：2026-04-30**

## 1) Overview / 概述
**EN:** WCL Combat-Time DPS Toggle ("the Extension") is a browser extension that recalculates DPS/WDPS/HPS display values on Warcraft Logs report pages using combat-time logic.

**中文：** WCL Combat-Time DPS Toggle（以下简称“本扩展”）是一个浏览器扩展，用于在 Warcraft Logs 报告页面中，按战斗时长口径重算 DPS/WDPS/HPS 显示值。

---

## 2) What We Collect / 我们收集什么
**EN:** The Extension does **not** collect personal information.

**中文：** 本扩展**不会**收集个人信息。

**EN:** The Extension only stores minimal preference data via `chrome.storage.sync`:
- `enabled`: whether combat-time recalculation is enabled
- `position`: floating widget position on the page

**中文：** 本扩展仅通过 `chrome.storage.sync` 保存最小化偏好设置：
- `enabled`：是否启用战斗时长重算
- `position`：页面悬浮面板位置

---

## 3) How Data Is Used / 数据用途
**EN:** Stored preference data is used only to provide extension functionality (toggle state and widget position persistence).

**中文：** 保存的偏好数据仅用于扩展功能本身（开关状态与面板位置持久化）。

---

## 4) Processing Scope / 处理范围
**EN:**
- Processing happens locally in your browser.
- The Extension runs only on matching pages: `*://*.warcraftlogs.com/reports/*`.
- No report content or personal data is sent to our servers.

**中文：**
- 所有处理均在你的浏览器本地完成。
- 扩展仅在匹配页面生效：`*://*.warcraftlogs.com/reports/*`。
- 不会将报告内容或个人数据发送到我们的服务器。

---

## 5) Permissions / 权限说明
**EN:**
- `storage`: used to store extension preferences (`enabled`, `position`).

**中文：**
- `storage`：用于保存扩展偏好设置（`enabled`、`position`）。

---

## 6) Data Sharing / 数据共享
**EN:** We do not sell, rent, or share user data with third parties.

**中文：** 我们不会向第三方出售、出租或共享用户数据。

---

## 7) Retention and Deletion / 保留与删除
**EN:**
- Preference data remains in browser extension storage until changed by you or until uninstall.
- After uninstall, extension data is removed according to browser behavior.

**中文：**
- 偏好数据会保留在浏览器扩展存储中，直到你修改或卸载扩展。
- 卸载后，扩展数据会按浏览器机制清理。

---

## 8) Security / 安全
**EN:** The Extension uses standard browser extension mechanisms and does not operate a custom remote data pipeline.

**中文：** 本扩展使用浏览器标准扩展机制，不运行自建远程数据通道。

---

## 9) Children's Privacy / 儿童隐私
**EN:** The Extension is not directed to children under 13 and does not knowingly collect children's personal data.

**中文：** 本扩展不面向 13 岁以下儿童，也不会有意收集儿童个人数据。

---

## 10) Policy Updates / 政策更新
**EN:** We may update this policy if extension behavior changes. The latest version will be published in the project repository.

**中文：** 如扩展行为发生变化，我们可能更新本政策。最新版本将发布在项目仓库中。

---

## 11) Contact / 联系方式
**EN:** For privacy-related questions, please open an issue in the project repository.

**中文：** 如有隐私相关问题，请在项目仓库提交 Issue。
